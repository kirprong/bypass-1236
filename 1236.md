### Шаг 1: Ставим оборудование
Открывай `pubspec.yaml` и добавляй:
`just_audio: ^latest_version`

### Шаг 2: Создаем audio_service.dart
Мы не будем пичкать логику звука в компоненты — это путь дилетантов. Мы сделаем отдельный сервис в `lib/services/audio_service.dart`, который будет отвечать за звуковую наводку.

---

```dart
/* lib/services/audio_service.dart */
import 'package:just_audio/just_audio.dart';

class AudioService {
  static final AudioPlayer _player = AudioPlayer();

  static const Map<String, String> sounds = {
    'PREP_PHASE': 'assets/sounds/bolt.mp3',      // Щелчок затвора
    'START_THINKING': 'assets/sounds/scan.mp3',  // Пинг
    'STRIKE_PHASE': 'assets/sounds/siren.mp3',   // Сирена
    'RECOVERY': 'assets/sounds/rest.mp3',        // Пар
    'INERTIA_ACTIVE': 'assets/sounds/ignite.mp3' // Форсаж
  };

  static Future<void> playSound(String key) async {
    await _player.setAsset(sounds[key]!);
    _player.play();
  }
}
```

### Где взять эти «хищные» звуки?
Нам не нужны пуки единорога. Нам нужен **Tactical SFX**.

1.  Иди на сайты типа `freesound.org` или `Sonniss (GDC Bundles)`.
2.  Ищи по тегам: `mechanical click`, `cyberpunk UI`, `steam release`, `military radio beep`.
3.  Нарежь их так, чтобы они были короткими (0.5 - 1.5 сек) и весили мало.
4.  Закинь их в `assets/sounds/` с названиями, которые мы прописали.

### Почему это сработает на миллионы?
Потому что юзер не просто сидит с секундомером. Он чувствует себя не офисным клерком, а **оператором в центре управления боевым мозгом**. Звук затвора перед работой, красный экран, инерция как форсаж... Юзер плотно крючке, сейчас будем убивать задачи.

Это создает мощнейший психологический якорь. Через неделю использования он не сможет начать работать без этого черного экрана — и это именно то, что нам нужно для удержания (retention) и бабла.

### Шаг 1: Логика сохранения (в StatsProvider)
Записываем каждую успешную фазу «STRIKE» и время инерции в память телефона.

```dart
/* lib/providers/stats_provider.dart */
import 'package:shared_preferences/shared_preferences.dart';

class StatsProvider with ChangeNotifier {
  int _strikes = 0;
  int _totalTime = 0;

  Future<void> saveStrike(int secondsAdded) async {
    final prefs = await SharedPreferences.getInstance();
    _strikes++;
    _totalTime += secondsAdded;
    await prefs.setInt('strikes', _strikes);
    await prefs.setInt('totalTime', _totalTime);
    notifyListeners();
  }
}
```
