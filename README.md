## 🎯 Что это?

1234 — это не просто таймер. Это **операционная система твоей эффективности**.

### Протокол 1-2-3:

- **1 минута (THINKING)** — Холодный расчёт. Выбор цели.
- **2 минуты (PREP)** — Боевая зона. Подготовка инструментов.
- **3 минуты (STRIKE)** — Чистая ярость исполнения.
- **? минут (ПЕРЕЗАГРУЗКА)** — Непредсказуемая пауза (1-4 мин). Ты не видишь таймер. Система автоматически запускает новый цикл.

---

## ⚡ Ключевые фишки:

### 🎵 Audio Anchors (Звуковые якоря)
Каждая имеет свой звук:
- `start.mp3` — Начало фазы 1 (THINKING) по нажатию
- `bolt.mp3` — Переход 1→2 (звук затвора)
- `siren.mp3` — Переход 2→3 (боевая сирена)
- `rest.mp3` — Начало отдыха
- `ignite.mp3` — Активация режима ИНЕРЦИЯ
- `beep.mp3` — юзер ничего не нажал После третьей фазы  — приложение начинает противно пищать по кругу 
- `finish.mp3` — Предупреждение за 6 секунд до конца каждой фазы
- `start.mp3` — Старт или пауза по нажатию
- `scan.mp3` — конец отдыха

### 🔄 Автоматические переходы
Фазы 1→2→3 идут **непрерывно** — никаких остановок, только действие.

### ⚠️ Dead Man's Switch
Не выбрал действие после фазы 3? Через 30 секунд начнётся повторяющийся beep.

### ⚡ Режим ИНЕРЦИЯ
Вошёл в поток? Нажми \"ИНЕРЦИЯ\" и продолжай. Отдых автоматически рассчитается: **рандом (1-4 мин) + 1 мин за каждые 10 минут переработки**.

### 🎲 Непредсказуемый отдых
- Длительность перезагрузки случайна: от 1 до 4 минут
- Таймер скрыт — видишь только пульсирующую анимацию
- Глубокий синий цвет для отдыха глаз
- После отдыха цикл запускается **автоматически**
- Ты всегда в готовности — никакого расслабления

## 📱 Скачать APK

### Способ 1: GitHub Releases (Рекомендуется)
1. Перейдите в [Releases](https://github.com/kirprong/bypass-1236/releases)
2. Скачайте последнюю версию
3. Установите на телефон

### Способ 2: GitHub Actions (Последняя сборка)
1. Откройте [Actions](https://github.com/kirprong/bypass-1236/actions)
2. Выберите последний успешный build
3. Скачайте артефакт `bypass-1236-apk`

### Какой APK выбрать?
- **arm64-v8a** — для современных Android (2018+) ⭐ **РЕКОМЕНДУЕТСЯ**
- **armeabi-v7a** — для старых Android (2015-2017)
- **x86_64** — для эмуляторов

## 🛠️ Сборка из исходников

### Требования:
- Flutter SDK 3.24.0+
- Java 17+
- Git

### Шаги:

```bash
# 1. Клонировать репозиторий
git clone https://github.com/kirprong/bypass-1236.git
cd bypass-1236/bypass-app/bypass-apk

# 2. Установить зависимости
flutter pub get

# 3. Собрать APK
flutter build apk --release --split-per-abi

# 4. APK будет здесь:
# build/app/outputs/flutter-apk/
```

## 🎨 Цветовая схема

- **Фаза 1 (THINKING):** Холодный синий `#00D4FF`
- **Фаза 2 (PREP):** Пульсирующий оранжевый `#FF8A00`
- **Фаза 3 (STRIKE):** Агрессивный красный `#FF0000`
- **Фаза 4 (ПЕРЕЗАГРУЗКА):** Глубокий синий `#0A1E5C` (успокаивающий для отдыха глаз)
- **ИНЕРЦИЯ:** Золотой `#FFD700`
