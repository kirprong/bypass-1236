## 🎲 Новая механика фазы отдыха (ПЕРЕЗАГРУЗКА)

### Что изменилось:

#### ⏱️ Рандомная длительность
- **Было:** Фиксированные 6 минут
- **Стало:** Случайная длительность от 1 до 4 минут
- Генерируется заново при каждом запуске фазы
- Пользователь не знает точное время отдыха

#### 🔒 Скрытый таймер
- **Было:** Показывался обратный отсчет времени
- **Стало:** Вместо таймера - пульсирующая анимация
- Иконка обновления (autorenew) с плавной пульсацией
- Анимация синхронизирована с дыханием (2 сек на цикл)

#### 🎨 Новый визуальный дизайн
- **Цвет:** Глубокий синий (#0A1E5C) вместо матричного зеленого (#00FF41)
- **Эффект:** Пульсирующее свечение и размытие
- **Цель:** Отдых для глаз, снижение напряжения
- **Название фазы:** "ПЕРЕЗАГРУЗКА" вместо "RECOVERY"
- **Текст:** "БУДЬ ГОТОВ ВСЕГДА" вместо "ПЕРЕГРУППИРОВКА"

#### 🔄 Автоматический перезапуск цикла
- **Было:** После отдыха таймер полностью сбрасывался, требовалось действие пользователя
- **Стало:** Автоматический переход к фазе 1 (THINKING) и немедленный запуск
- Непрерывный рабочий цикл без пауз
- Пользователь всегда в готовности

## 🎯 Преимущества новой системы

### 1. Психологические
- ❌ **Убирает предсказуемость** - мозг не может "досидеть последние секунды"
- ⚡ **Держит в тонусе** - всегда готов к возобновлению работы
- 🧘 **Настоящий отдых** - не отвлекаешься на таймер
- 🎯 **Фокус на процессе** - не на ожидании конца отдыха

### 2. Физиологические
- 👀 **Отдых для глаз** - глубокий синий цвет успокаивает
- 🌊 **Ритмичность** - пульсация синхронизируется с дыханием
- 💆 **Расслабление** - отсутствие визуального стресса

### 3. Практические
- 🔁 **Непрерывность** - автоматический перезапуск цикла
- 📊 **Гибкость** - переменное время отдыха адаптируется к потребностям
- 🎮 **Геймификация** - элемент неожиданности

## 🎮 Формула расчета отдыха

### Обычный режим (после фазы 3):
```
Отдых = Random(60, 240) секунд
      = Random(1, 4) минут
```

### Режим ИНЕРЦИЯ (после переработки):
```
Базовый отдых = Random(60, 240) секунд
Бонус = (inertiaSeconds / 600) * 60 секунд
Итого = Базовый отдых + Бонус

Пример:
- Инерция: 20 минут (1200 сек)
- Базовый отдых: 180 сек (рандом)
- Бонус: (1200 / 600) * 60 = 120 сек (2 мин)
- Итого: 180 + 120 = 300 сек (5 мин)
```
