### 2. Обновить AudioService
В `lib/services/audio_service.dart` оберните всё в try-catch:

```dart
Future<void> initialize() async {
  if (_isInitialized) return;

  try {
    // Весь код инициализации
    _isInitialized = true;
  } catch (e) {
    debugPrint('AudioService init failed (SAFE MODE): $e');
    _isInitialized = false; // Работаем без звука
  }
}

Future<void> playSound(String key) async {
  if (!_isInitialized) {
    debugPrint('AudioService not initialized - skipping sound');
    return; // Просто пропускаем
  }
  // остальной код
}
```
