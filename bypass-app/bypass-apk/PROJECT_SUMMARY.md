## 🎯 Реализованные функции

### ✅ Таймер 1-2-3-6
- **Phase 1 (60 сек)** - THINKING: Анализ задачи
- **Phase 2 (120 сек)** - PREP: Подготовка
- **Phase 3 (180 сек)** - STRIKE: Работа
- **Phase 4 (360 сек)** - RECOVERY: Отдых

### ✅ Режим INERTIA (God Mode)
- Активируется во время фазы STRIKE
- Безлимитный таймер вверх
- Автоматический расчет дополнительного отдыха
- Золотой цвет интерфейса

### ✅ Система рангов
- **ПЛАНКТОН** - 0 strikes
- **БОЕЦ** - 100 strikes
- **ВОИН** - 200 strikes
- **ХИЩНИК** - 300 strikes
- **APEX PREDATOR** - 500 strikes

### ✅ Звуковое сопровождение
- scan.mp3 - Старт Phase 1 (THINKING)
- bolt.mp3 - Старт Phase 2 (PREP)
- siren.mp3 - Старт Phase 3 (STRIKE)

## 🔊 Звуковые якоря (Audio Anchors)

Каждое событие в приложении имеет свой уникальный звук для создания условных рефлексов:

| Событие | Файл | Когда играет | Описание |
|---------|------|--------------|----------|
| **Старт/Пауза** | `start.mp3` | При нажатии START/PAUSE | Начало или пауза таймера |
| **Переход THINKING → PREP** | `bolt.mp3` | Фаза 0 → 1 (через 60 сек) | Звук затвора |
| **Переход PREP → STRIKE** | `siren.mp3` | Фаза 1 → 2 (через 120 сек) | Боевая сирена |
| **Начало отдыха** | `rest.mp3` | Начало фазы RECOVERY | Релаксация |
| **Предупреждение** | `finish.mp3` | За 6 сек до конца фазы 0,1,3 | Подготовка к переходу |
| **Активация инерции** | `ignite.mp3` | Нажатие кнопки OVERDRIVE | Режим инерции |
| **Dead Man's Switch** | `beep.mp3` | Каждую секунду после фазы 2 | Противное пищание (если нет выбора) |
| **Конец отдыха** | `scan.mp3` | Завершение цикла | Окончание фазы RECOVERY |

### Логика воспроизведения:
1. **Нажатие START** → `start.mp3`
2. **Через 60 сек** (фаза 0 → 1) → `bolt.mp3`
3. **За 6 сек до конца фазы 0,1,3** → `finish.mp3`
4. **Через 120 сек** (фаза 1 → 2) → `siren.mp3`
5. **Завершение фазы 2** → ждём выбора (ИНЕРЦИЯ или ОТДЫХ)
6. **Если нет выбора 30 сек** → `beep.mp3` каждую секунду (Dead Man's Switch)
7. **Выбор ОТДЫХ** → `rest.mp3` → начало фазы 3
8. **Выбор ИНЕРЦИЯ** → `ignite.mp3` → режим овердрайва
9. **Конец отдыха** → `scan.mp3` → цикл завершён
